# 💰 Personal Expense Tracker (SpendSense Tracker)

A command-line based expense management application developed in Python, designed to record, organize, and analyze daily financial transactions. This project emphasizes structured programming, persistent storage, and practical financial insights.

---

## 📌 Overview

SpendSense Tracker is a lightweight CLI tool that enables users to track expenses efficiently while gaining meaningful insights into their spending patterns. It combines simple data storage with analytical features such as budgeting, categorization, and visualization.

---

## 🚀 Key Features

- 📥 Record expenses with structured fields (amount, category, note, date)  
- 💾 Persistent storage using JSON for data reliability  
- 📊 Total spending calculation and category-wise summaries  
- 📈 Spending insights (highest, lowest, and average expenditure)  
- 💸 Budget tracking with percentage usage  
- ⚠️ Budget alerts for nearing and exceeding limits  
- 🔄 Dynamic budget modification with persistent storage  
- 📅 Date-based filtering for targeted analysis  
- 📉 Data visualization using bar charts (Matplotlib)  
- 🗑️ Safe bulk deletion with confirmation prompt  

---

## 🛠️ Tech Stack

- 🐍 Python 3  
- 🧾 JSON (lightweight data persistence)  
- 📊 Matplotlib (data visualization)  

---

## ▶️ Execution

```bash
pip install matplotlib
python expense_tracker.py
```
