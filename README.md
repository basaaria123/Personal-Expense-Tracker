# Personal-Expense-Tracker

A command-line based expense management application developed in Python, designed to record, organize, and analyze daily financial transactions. This project emphasizes structured programming, persistent storage, and basic data visualization.

## Key Features

* Capture expenses with structured fields (amount, category, note, date)
* Persistent data storage using JSON
* Aggregate insights including total spending and category-wise distribution
* Date-based filtering for targeted analysis
* Data visualization via bar charts using Matplotlib
* Safe bulk deletion with confirmation prompt

## Tech Stack

* Python 3
* JSON (lightweight data persistence)
* Matplotlib (data visualization)

## Execution

```bash id="u4u7x2"
pip install matplotlib
python expense_tracker.py
```

* Implements modular function-based design
